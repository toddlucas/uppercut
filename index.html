uppercut by toddlucas

Uppercut is a Backbone style model transport that's compatible with Knockout.

For projects that need a way of syncing client model state with the server, Uppercut is a simple solution. The core sync method is derived directly from Backbone in order to provide server compatibility. Use Uppercut with Knockout for great justice.
